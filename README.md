# Installation

Add the dependency:

```ruby
npm i @paraboly/react-native-info-card
```

## Peer Dependencies

###### IMPORTANT! You need install them

```js
"react": ">= 16.x.x",
"react-native": ">= 0.55.x",
"react-native-androw": ">= 0.0.34",
```

# Usage

## Import

```js
import InfoCard from "@paraboly/react-native-info-card";
```

## Component1 Usage

```js
<InfoCard />
```
